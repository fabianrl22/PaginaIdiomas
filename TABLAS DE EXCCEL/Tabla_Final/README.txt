Para subir los archivos csv al PHPMyAdmin se sigue los siguientes pasos:

1. Del excel se guarda el archivo con formato csv UTF8 delimitado por comas
2. En phpMyAdmin se selecciona import y se elige el archivo 
3. Conjunto de caracteres del archivo:  UTF-8
4. Formato CSV using load data
5. Columnas separadas por: ;
   columnas encerradas por: vacio
   caracter de escape: \

   No se toca nada mas
